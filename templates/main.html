О системе

Данный сервис предназначен для ведения учета потерь военной техники в ходе боевых действий. Система позволяет собирать, анализировать и визуализировать данные по ликвидированной технике.

Статистика

Вы можете просматривать статистику уничтоженной техники по различным параметрам: по регионам, по датам и по типам техники. Данные можно выгрузить в отчеты.
